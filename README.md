## 2. 역할 분담

### 🐰 김지환

- **방명록 기능 구현 (Firebase 연동)**
    - 방명록 페이지의 이름 입력란 / 내용 입력란 / 완료 버튼 / 닫기 버튼 구현, 방명록 등록 시에 알림창
    - 메인 페이지에서 방명록 열람 위한 데이터베이스 연동, 방명록 페이지로 넘어가는 팝업창 구현
- **QnA 기능 구현 (Firebase 연동)**
    - 질문 입력란 / 등록 버튼 구현, 등록 시에 알림창 / 답변하기 버튼 구현
    - 답변 시에 비밀번호 입력창 / 입력 시에 알림창 구현
- **CSS 코드 정리 / 폴더화**
- **발표 PPT 제작**

<br>
    
### 🐴 강동연

- **이벤트 핸들링 구현**
    - 메인 페이지의 와이어프레임 / 기본 뼈대 제작
    - 팀원 이미지의 hover 기능 활성화 시에 클릭 / 상세페이지 팝업창 구현
- **팀원 소개**
    - 팀원 상세 페이지로 넘어가기 위한 hover 기능 활성화, 이미지 위에 hover 기능 구현
- **README 제작 / 편집**

<br>

### 🐷 김하정

- **메인 페이지 구성**
    - 메인 페이지의 기본 뼈대 세부화 / 구성 제작
    - 전체적인 CSS 기본 틀 제작
- **기능 코드 병합**
    - 각 팀원의 코드 병합 / 정리 / 구현
- **README 제작 / 편집** 

<br>

### 🐲 김채원

- **팀원 소개 세부사항 페이지**
    - 팀원 상세 페이지의 CSS 제작, 팀원 상세 페이지의 구성 / 내용 제작
    - 블로그 링크 기능 구현
    - 이미지 움직이는 동작 기능 구현
- **README 제작 / 편집 / gif 영상 제작**

## 10. 프로젝트 후기

### 🐰 김지환

이전에 프로젝트를 진행할 때 대부분 백엔드 부분에서 맡은 기능만 수행했었지만 이번 기회에 팀장을 맡아 프로젝트를 체계적으로 관리하는 능력을 키울 수 있었습니다. 특히 팀원들과 협력하여 각각의 역할을 명확히 하고,진행사항을 점검하면서 목표 달성을 할 수 있었습니다.
또한,팀원들의 다양한 의견을 수렴하고 회의를 하면서 팀워크도 키울 수 있었습니다. 첫 프로젝트라 걱정이 있었지만 좋은 팀원들을 만나 수월하게 마무리 돼서 좋았습니다.

<br>

### 🐴 강동연

팀소개 페이지를 만드는 과정에서 이전에 배운 스파르타코딩 데이터를 들고와서 그냥 대충하면 되겠지 라며 가볍게 팀프로젝트에 참여 했었는데 팀원 과의 소통을 통해 열정적인 모습을 볼 수 있었으며 스스로 만든 작업물에 비해 더 나은 작업 결과물 들이 나와서 내심 부끄러운점도 많았습니다 혼자 였으면 이런 결과물이 나오지 않았을 것 같으며, 소통을 통해 어려움을 잘 해결할 수 있게 되었고 같이 즐겁게 작업해서 좋았습니다.

<br>

### 🐷 김하정

첫 팀 프로젝트인데 아직 배운 기술도 모두 익숙하지 않아서 처음에는 뭐부터 어떻게 해야 하나 조금 막막했는데, 팀원 중에서 누군가 어떤 의견을 제시하면 그 의견을 세부화 하면서 틀이 잡혀가는 기분이 들었습니다. 그렇게 계속 혼자가 아니라 팀원분들이랑 함께 웃기도 많이 웃고, 회의하고, 서로 도와가며 역할 분담도 하여 작업을 하다 보니 오히려 나중에는 작업해야 할 코드가 한가득이라 시간이 부족하게 느껴졌습니다.
팀원분들 덕분에 재밌게 작업 하다보니 집중력도 높아졌고, 내가 원하는 기능을 구현하고자 하는 욕심도 많아져서 학습 시간 내내 굉장히 좋은 자극이 된 것 같습니다. 착하고 성실하고 재능 충만한 우리 팀원분들 감사해요 ㅋㅋ🫶
